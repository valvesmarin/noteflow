# Idea del Proyecto - NoteFlow

## ¿Qué problema resuelve NoteFlow?

NoteFlow resuelve el problema de tener notas, listas de tareas e ideas rápidas dispersas entre diferentes aplicaciones (WhatsApp, Notas del celular, papel, Google Keep, etc.). Esto genera desorganización, pérdida de información y falta de seguimiento.

NoteFlow centraliza todo en un solo lugar: notas de texto, checklists y notas de ideas rápidas, con una interfaz limpia, rápida y pensada para el uso diario en el celular.

## ¿Quién es el usuario objetivo?

- Estudiantes que organizan apuntes, tareas y brainstormings.
- Profesionales que gestionan múltiples proyectos y listas pendientes.
- Personas que quieren tener todas sus notas, tareas e ideas en un solo aplicativo bonito y rápido.
- Usuarios que prefieren una experiencia nativa (no solo una web).

Uso diario: abrir la app varias veces al día para anotar algo rápido, marcar tareas como completadas o capturar ideas.

## Funcionalidades principales (MVP - Versión 1.0)

- Tres tipos de notas:
  - Notas de texto normales
  - Checklists / Listas de tareas
  - Notas de ideas rápidas (con tags y colores)
- Crear, editar y eliminar notas
- Persistencia local con AsyncStorage
- Modo claro y oscuro automático
- Navegación por pestañas (Tabs) con Expo Router
- Listas de alto rendimiento con FlashList
- Validación de formularios con Zod
- Búsqueda en tiempo real
- Diseño moderno y limpio con Gluestack UI

## Funcionalidades opcionales (futuras)

- Sincronización en la nube
- Recordatorios y notificaciones push
- Archivar notas en lugar de eliminar
- Colaboración entre usuarios
- Estadísticas de productividad
- Exportar notas en PDF
