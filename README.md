# NoteFlow 📝

**A modern mobile app to organize notes, checklists and quick ideas**

NoteFlow is a native mobile application built with **React Native + Expo** that allows users to manage three types of content in one place: text notes, task checklists, and quick idea notes with tags and colors.

---

## ✨ Features

- **Text Notes**: Create, edit and delete notes with title and content
- **Checklists**: Task lists with the ability to mark items as completed
- **Quick Ideas**: Notes with tags and color selection for brainstorming
- Local data persistence with AsyncStorage
- Global state management with **Zustand**
- Tab navigation using **Expo Router**
- Clean and modern UI with **React Native Paper**

---

## 🛠 Technologies

- **React Native** + **Expo**
- **Expo Router** (file-based routing)
- **TypeScript**
- **Zustand** + AsyncStorage persistence
- **React Native Paper** (UI components)
- **React Native** core components
